 #  Smart Character Counter

A modern and responsive character counter built using **HTML, CSS, and JavaScript**.

##  Features

-  Live character counting
-  Remaining characters display
-  Warning when limit is near
-  Input limit enforcement
-  Smooth shake animation on limit exceed
-  Custom popup modal (reset option)
-  Clean and modern UI
-  Fully responsive design

---

##  Preview

> Type in the textarea and see live updates with animations.

---

##  Technologies Used

- HTML5
- CSS3 (Flexbox + Animations)
- JavaScript (DOM Manipulation)

---

##  How It Works

- Tracks user input in real-time
- Limits input to **60 characters**
- Shows remaining characters
- Adds warning styles when nearing the limit
- Triggers animation + popup when limit is exceeded
